## 问答题

填写表格。需要利用一系列的单位：$K = 2^{10}, M=2^{20}, G=2^{30}, T=2^{40}, P=2^{50}, E=2^{60}$

| 虚拟地址位数 | 虚拟地址数                             | 最大可能得虚拟地址         |
| :----- | --------------------------------- | ----------------- |
| $8$    | $2^8=16$                          | $15$              |
| $16$   | $2^{16} = 64 K=2^6 \times 2^{10}$ | $2^{16}-1=64K-1$  |
| $32$   | $2^{32}$                          | $2^{32}-1=4G-1$   |
| $48$   | $2^{48} = 256T=2^8\times 2^{40}$  | $2^{48}-1=256T-1$ |
| $64$   | $2^{64}=16E$                      | $2^{64}-1=16E-1$  |
